Requirements

zIndex 

var zIndex: Int

Tool's interaction view z-Index in superview. The higher the value, the top the view.

You can use ADInteractZIndex.rawValue or custom int value.

System line draw tool's zIndex is ADInteractZIndex.Bottom.rawValue+1, mosaic draw tool's zIndex is ADInteractZIndex.Bottom.rawValue. System sticker tool's zIndex is ADInteractZIndex.Top.rawValue

strategy 

var strategy: ADInteractStrategy

Tool's interaction view response strategy.

interact​Clip​Bounds 

var interactClipBounds: Bool

Whether interaction view clip to clip rect bounds when interacting.

Under normal circumstances, the size of the interactive view and the image are the same. Returning false means that the interactive area can exceed the boundary of the view during interaction, and vice versa.
